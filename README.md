**TestNG RestAssured Automation**
A simple API testing framework using Java, TestNG, and RestAssured.
It includes basic test cases, a TestNG suite, Maven setup, and auto‑generated reports for quick and reliable API validation.

Project contents
src/test — API test classes
pom.xml — Maven dependencies
testng.xml — Test suite config
test-output — TestNG HTML reports

Run tests
From IDE: run testng.xml
